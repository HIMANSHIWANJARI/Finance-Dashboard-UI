
# Finance Dashboard

A modern and responsive **Finance Dashboard** built using React and SCSS to help users track income, expenses, and overall financial performance in a simple and visual way.

---

## Features

*  **Dashboard Overview**

  * Displays Total Balance, Income, and Expenses
  * Visual charts for financial trends

* **Transactions Management**

  * View all transactions
  * Search, filter, and sort functionality
  * Categorized data (Food, Travel, Shopping, etc.)

*  **Role-Based UI**

  * **Viewer**: Read-only access
  * **Admin**: Add, edit, and delete transactions

*  **Insights Section**

  * Highest spending category
  * Income vs Expense comparison
  * Total savings calculation

*  **UI/UX**

  * Clean and modern dashboard design
  * Fully responsive (mobile, tablet, desktop)
  * Styled using SCSS for better structure and maintainability

---

##  Tech Stack

*  React (JavaScript)
*  SCSS (Styling) and Tailwind CSS
*  Recharts / Chart.js (for data visualization)
*  Context API (State Management)

## Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/HIMANSHIWANJARI/Finance-Dashboard-UI.git
```

2. Navigate to the project folder:

```bash
cd finance-dashboard
```

3. Install dependencies:

```bash
npm install
```

4. Run the project:

```bash
npm run dev
```
